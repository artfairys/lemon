如何将青柠起始页设置成浏览器主页 / 新标签页？
两种方法：
1. 在浏览器设置中将启动页和主页设置为 https://limestart.cn；
2. 如果您使用的是电脑 Chrome / Edge / Firefox 浏览器，可以直接安装青柠起始页扩展，安装后浏览器新标签页会被自动替换成青柠起始页。安装地址：Chrome 版扩展 / Edge 版扩展 / Firefox 版扩展。
为什么无法把青柠起始页设置成浏览器主页？
大概率是您安装的杀毒软件 / 电脑管家（特别是联想电脑管家）导致的。请检查杀毒软件 / 电脑管家的设置里是否把浏览器主页锁定成别的了？
青柠起始页经常莫名其妙地自动刷新？
如果您从别的网页切换回青柠起始页时 / 在青柠起始页打开浏览器收藏夹里的网站时 / 进行其它一些操作时，青柠起始页会自动刷新：请关闭您安装的浏览器扩展插件，逐一排查；已知 IDM 扩展插件会导致上述问题。
为什么按下回车无法搜索 / 同时打开两次搜索结果页面？
1. 可能是您安装的浏览器扩展冲突了，可以把扩展都关掉逐个排查；
2. 可能是您的浏览器没有开启「允许弹出窗口」权限，打开就行啦。
每次重启浏览器都要重新登录账号？
可能是因为浏览器开启了「退出时清理浏览痕迹」或者没有允许青柠起始页的 Cookie 权限，把权限打开就会自动登录了。
预设的捷径可以删除吗？
可以，在图标上单击鼠标右键 / 触摸长按即可弹出删除选项。折叠其它预设捷径并重新展开后，所有已删除的预设捷径便会还原；如果所有预设捷径都已被删除，请点此还原。
如何在「网站捷径」页面和「便笺」页面之间快速切换？
在青柠起始页二级界面【左右两侧的空白处】滚动鼠标滚轮/按键盘左右方向键（电脑）或双指滑动（移动设备和笔记本触控板）即可快速切换两个页面。
每次打开网站捷径页都要点击「时间」，有更快捷的方法吗？
在页面空白处直接单击鼠标右键即可。或者在「常规设置」里开启「青柠起始页加载时自动显示二级界面」。
界面动画卡顿掉帧很严重？
可以在「常规设置」里开启「流畅模式」或「减弱动态效果」。
不喜欢新版的二级界面，如何回到旧版界面？
请点此前往青柠起始页 | 经典版。
搜索栏周围有一圈黑边？
请尝试关闭您显示器的 HDR 模式。
